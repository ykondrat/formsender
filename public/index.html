HTML Example

Setup the Your HTML form

Set your form's action-attribute to https://tranquil-sierra-86540.herokuapp.com/example@email.com and replace example@email.com with your own email.

Add a name-attribute to every field

Ensure that all of input, select and textarea tags inside your form have a name attribute.

Submit the form

Go to your website and submit the form, and you will receive email with form data.

Form example

<form method="POST" action="https://tranquil-sierra-86540.herokuapp.com/example@email.com">

    <input type="text" name="user_name">

    <input type="email" name="user_email">

    <input type="submit" value="Send">

</form>

JS Example

Setup the Your JS code

Set your request's url to https://tranquil-sierra-86540.herokuapp.com/example@email.com and replace example@email.com with your own email.

Set mathod

Ensure that the method of request's set to POST.

Set data type

Ensure that the property dataType of request's set to json.

Request example

$.ajax({

    url: 'https://tranquil-sierra-86540.herokuapp.com/example@gmail.com",

    method: 'POST',

    data: {

        name: 'John',

        surname: 'Smith'

    },

    dataType: 'json'

});
